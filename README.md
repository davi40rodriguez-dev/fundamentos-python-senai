# 🐍 Fundamentos de Python - SENAI

Este repositório reúne os exercícios desenvolvidos durante o curso **Fundamentos de Python** realizado no SENAI.

## 📚 Conteúdos estudados

- Variáveis
- Tipos de dados
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `elif` e `else`)
- Laços de repetição (`for` e `while`)
- Funções
- Listas
- Tuplas
- Dicionários
- Manipulação de dados
- Exercícios práticos

## 🎯 Objetivo

Registrar minha evolução na linguagem Python durante o curso do SENAI, mantendo todos os exercícios organizados para estudo e consulta futura.

## 🛠️ Tecnologias utilizadas

- Python 3
